Grok API Anahtarı Gerekli

Yapay zeka asistanı için xAI Grok API anahtarı girmeniz gerekiyor.
Bu işlemi yalnızca bir kez yapmanız yeterli.

  1. console.x.ai adresine gidin
  2. X (Twitter) veya Google ile giriş yapın
  3. Billing → kredi kartı ekleyin
  4. API Keys → Create API Key → chat:write izni seçin
  5. Anahtarınız xai- ile başlar

📤 CRM'e Aktar

Kaydedilen görüşmeleri saha_crm_*.json formatında indirin.
CRM → Görüşmeler sekmesi → JSON Yükle ile aktarın.

Nasıl çalışır?
  1. Bu uygulamada görüşmeleri kaydedin
  2. "JSON İndir" ile dosyayı indirin
  3. WhatsApp / Drive ile PC'ye gönderin
  4. CRM → Görüşmeler → "JSON Yükle" butonuna tıklayın
  5. Görüşmeler otomatik eklenir
